# St8tionZero

South African digital studio — websites, SEO, social media. Launch from zero.

**Live site:** [st8tionzero.co.za](#) (coming soon)

## Stack

- [Next.js 15](https://nextjs.org) (App Router, TypeScript)
- [GSAP 3](https://gsap.com) + ScrollTrigger — all key animations
- [@phosphor-icons/react](https://phosphoricons.com)
- Tailwind CSS (base reset only — design uses CSS custom properties)

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Standalone preview

Open `index.html` directly in any browser — no install needed. All dependencies load from CDN.

## Deploy

Connects to [Vercel](https://vercel.com) with zero config — import this repo and it builds automatically.
